# java-swing-sum

Aplicação desktop simples em Java Swing que soma dois números inteiros digitados pelo usuário.

## Funcionalidades
- Soma N1 + N2 ao clicar no botão "="
- Validação de entrada:
  - Campo vazio: mensagem específica (Preencha N1 / Preencha N2)
  - Texto não numérico: mensagem "Número inválido"

## Tecnologias
- Java (Swing)
- Eventos (`ActionListener`)

## Como executar
1. Abra o projeto na IDE (IntelliJ, Eclipse, etc.)
2. Execute a classe `TelaSoma`

## Estrutura
- `src/` (código-fonte)
